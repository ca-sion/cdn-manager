# Technology Stack

## Core Technologies
- **PHP 8.2+:** The primary programming language for backend logic.
- **Laravel 11:** The web application framework providing core services and architecture.
- **Filament v3:** The TALL stack-based admin panel framework for managing the backend interface.

## Frontend
- **Tailwind CSS:** Utility-first CSS framework for styling.
- **Vite:** Build tool and development server.
- **Livewire & Alpine.js:** Used via Filament for reactive UI components.

## Specialized Libraries
- **genkgo/camt:** Library for parsing CAMT bank statement files.
- **sprain/swiss-qr-bill:** Library for generating Swiss QR-bills.
- **spatie/laravel-media-library:** Manages file uploads and media associations for models.
- **rap2hpoutre/fast-excel:** For efficient Excel import/export operations.

## Testing & Quality
- **Pest:** Functional and unit testing framework.
- **Laravel Pint:** PHP code style fixer for maintaining consistent formatting.
